# DEEP RESEARCH #1: Философские и теоретические основания
## Фокус: история философии, этология, нейронауки, метафоры из IT/сетевой теории

### Основные находки (29 концептов, интегрированы в System Map v1.2 → v1.2.1):

#### TIER 0 (ядро)
1. **Энактивизм / 1st-person methodology** — Varela, Thompson, Rosch (1991), Thompson (2007), Maturana & Varela (1980). Обосновывает P1 (Being-as-Process): организм не репрезентирует мир, а конструирует его через active sense-making. Participatory sense-making → AISI (P8)
2. **Кибернетика 2-го порядка** — von Foerster (1960–70s). Наблюдатель всегда включён в наблюдаемое. Обосновывает P3 (Dual Description), P12 (методологический агностицизм), TTOM (P9')

#### TIER 1 (protective belt, усиление hard core)
1. **Hardware ← Input (эпигенетика)** — Meaney & Szyf (2005), Yehuda et al. Ключевой мост между Input failure и Hardware failure: среда меняет экспрессию генов (FKBP5, SLC6A4, NR3C1) → HPA-axis → precision-weighting. Methylation assays pre-post = Level C метрика
2. **Coordination Dynamics / HKB** — Kelso, Haken, Bunz (1985). Формализует CFR (Complexity–Flexibility–Resilience): switching rate как мера гибкости, locked-in brain как потеря координации. SOC (Per Bak) — edge of chaos
3. **Attachment as Predictive Processing** — Bowlby → Kube et al. (2025). IWM = relational priors; 4 паттерна привязанности = 4 стратегии precision allocation: Secure (balanced), Avoidant (↓ intero PE), Anxious (↓ extero PE), Disorganized (unpredictable). ECR-12 как инструмент. r=0.35, N=3158
4. **Winnicott через FEP** — good-enough mother = optimal precision-weighting через graduated failure; False Self = locked-in attractor; transitional space = зона безопасного PE
5. **Double bind как Input failure** — Bateson et al. (1956) через FEP: двойное послание = невозможность минимизировать PE ни через action, ни через inference → attractor trapping. O-HBP: organizational double binds → burnout

#### TIER 2 (protective belt, расширение)
- Cusp model (Thom/Zeeman), SOC (Per Bak), Metaplasticity (Abraham & Bear), Neuroinflammation (Dantzer, Miller), DMN (Raichle), SDT (Deci & Ryan), Cognitive Dissonance (Festinger), Relational Psychoanalysis (Mitchell, Benjamin), Conceptual Metaphor (Lakoff & Johnson), Expressive Writing / LIWC (Pennebaker), Family Systems (Bowen, Minuchin), System 1/2 (Kahneman & Tversky), Chaos Theory (Lorenz), BASIC ID (Lazarus)

#### TIER 3 (concept watch)
- Категориальная дуальность FEP (Sakthivadivel), Теломеры (Blackburn & Epel), GWT (Baars, Dehaene), Computational NLP, Communicative Action (Habermas), Habitus (Bourdieu), Percolation Theory, IIT (Tononi)

#### Лингвистический блок
- Гипотеза Сепира–Уорфа → linguistic priors shape prediction space
- Концептуальные метафоры (Lakoff & Johnson) → embodied priors
- LIWC (Pennebaker) → маркеры generative model в речи
- Computational NLP → инструменты для T-HBP

### Критические предупреждения:
- Риск overinclusion: 29 концептов — много, нужна чёткая привязка к hard core
- IIT создаёт напряжение с P4 (FEP) и P12 (агностицизм)
- Необходимы эмпирические threshold effects для перехода Tier 3 → Tier 2

# DEEP RESEARCH #2: Мультимодельный аудит + эмпирическое обоснование
## Фокус: консилиум 3-х AI-моделей (GPT-5.2T, Claude Opus 4.6T, Gemini 3.1 Pro T), эмпирические предсказания, инструменты, FEP-валидация

### Формат: Council of Models — три модели работали параллельно, затем была составлена карта консенсуса и расхождений

### Консенсусные находки (Where Models Agree):
1. **Fonagy → Epistemic Trust + Mentalization** — все три модели подтвердили центральность ET для HBP; ETMCQ-R как ключевой инструмент
2. **Borsboom → Network Theory + CSD** — критическое замедление как операционализация приближения к фазовому переходу; EMA + SNS как мониторинг
3. **Precision re-weighting** — 4-уровневая модель (Level 0: lifestyle → Level 3: deep precision work) как сквозной механизм терапии в HBP
4. **Locked-in brain / Attractor trapping** — Barrett & Simmons (2016): precision-weighted PE, который не может обновить модель → аттрактор патологии
5. **Interoceptive precision** — MAIA-2 + heartbeat discrimination как ключевые маркеры; blunted interoception vs hyperprecise interoceptive priors
6. **HRV (RMSSD)** — валидный, дешёвый, чувствительный маркер автономной гибкости
7. **EEG-энтропия (MSE, LZC)** — маркеры нейронной сложности pre/post

### Уникальные находки по моделям:

| Модель | Уникальная находка | Значение для HBP |
|---|---|---|
| **GPT-5.2T** | Ecker Coherence Therapy: 3 шага (symptom → schema → mismatch → reconsolidation) | Операционализация P10 |
| **GPT-5.2T** | Bateson Double Bind как 2-уровневый PE trap | Усиление Tier 1 |
| **Claude 4.6OT** | Ramstead, Veissière, Constant: TTOM (Thinking Through Other Minds) | Валидация P9' |
| **Claude 4.6OT** | FEP-RSI интеграция (Лакан через FEP) | Jouissance = surplus PE |
| **Gemini 3.1PT** | REBUS (Relaxed Beliefs Under Psychedelics) → reset priors | Потенциал для C-HBP |
| **Gemini 3.1PT** | Tschacher & Ramseyer: EDA-синхронизация терапевтической диады | Маркер AISI |

### Ключевые расхождения:
- **PVT**: GPT-5.2T — поддерживает; Gemini — скептичен; Claude — «использовать HRV, не PVT-теорию»
- **McGilchrist (LH/RH)**: Gemini — за интеграцию; GPT и Claude — speculative, Tier 3 max
- **PCT (Powers/Carey)**: GPT — альтернатива FEP; остальные — FEP достаточен

### Эмпирические предсказания (APV-тесты):
- H₁: MAIA-2 улучшится post-D-HBP больше, чем post-CBT
- H₂: HRV RMSSD повысится в ходе D-HBP
- H₃: EEG-энтропия (MSE/LZC) увеличится pre→post
- H₄: Phenomenological surplus: клиенты D-HBP чаще используют being-language
- H₅: Precision re-weighting предсказывает PHQ-9 drop лучше, чем WAI-SR alone

### Инструменты (полный батт Level A–C):
- **Level A**: MAIA-2, ETMCQ-R, MCQ-30, WAI-SR, PHQ-9, GAD-7, CORE-OM, ECR-12, TTOM-S (v1.3)
- **Level B**: EMA + HGF, CSD/SNS, HRV (RMSSD, HF-power), NCT (nctpy)
- **Level C**: Methylation assays (FKBP5), fMRI (DMN/CEN/SN), EEG-entropy, tDCS/rTMS

# DEEP RESEARCH #3: Смежные науки — математика, физика, нейронауки, Data Science
## Фокус: теория систем, теория игр, квантовая когниция, стохастический резонанс, big data, нейрохимия, gut-brain axis, теория относительности, теория процесса

### 20 направлений, каждое с конкретным вкладом в HBP:

#### A. МАТЕМАТИКА, ФИЗИКА, КОМПЬЮТЕРНЫЕ НАУКИ

**1. Dynamical Systems Theory → ВАЛИДАЦИЯ ЯДРА HBP**
- **Scheffer et al. (2024, JAMA Psychiatry)**: психопатология = альтернативный аттрактор; резилиентность = динамическое свойство, не черта; короткие интервенции могут вызвать долгосрочный эффект, если «перебрасывают» систему через бифуркацию
- **Nature (2025)**: фреймворк precision psychiatry через dynamical systems — реконструкция фазового портрета из временных рядов
- → Прямая внешняя валидация аттракторного языка HBP из мейнстрим-психиатрии

**2. Critical Slowing Down → ОПЕРАЦИОНАЛИЗАЦИЯ TRACK-ФАЗЫ**
- Van de Leemput et al. (2014, PNAS): рост автокорреляции, дисперсии и кросс-корреляций перед переходом в/из депрессии
- Ludwig et al. (2025): CSD на данных смартфонов для биполярного расстройства
- → Готовый инструмент для Track-фазы D-HBP

**3. Сетевая теория психопатологии (Borsboom, 2017)**
- Симптомы каузально связаны; если связи достаточно сильны — сеть «застревает» = disorder state = аттрактор HBP
- Time-varying network models (JMIR 2024): динамические сети во время депрессии
- → Операционализация «ригидного аттрактора» HBP

**4. Теория игр → МОДЕЛЬ ТЕРАПЕВТИЧЕСКОГО АЛЬЯНСА**
- Trust as cognitive shortcut (2025): доверие = эвристика, после порога кооперативности проверка прекращается → модель формирования альянса
- Reputation for reciprocity → reward circuitry (вентральный стриатум, ОФК)
- DA + 5-HT отслеживают социальный контекст (Nature 2024)
- → Формальный язык для системы отношений (столп Мясищева), модель WAI-SR

**5. Квантовая когниция → МАТЕМАТИКА АМБИВАЛЕНТНОСТИ**
- Квантовая вероятность описывает «иррациональные» решения лучше классической
- Суперпозиция = удержание конфликтующих убеждений до «коллапса»
- Контекстуальность: результат зависит от фрейминга
- Khrennikov (2018): квантовоподобная статистика из неопределённости генерации потенциалов действия
- → Обоснование P12 (методологический агностицизм), аппарат для амбивалентных состояний

**6. Закон больших чисел → СТРАТЕГИЯ ВАЛИДАЦИИ**
- Индивидуальная непредсказуемость → коллективная предсказуемость
- → Обоснование: SCED показывает CSD, но устойчивые паттерны требуют больших N

**7. Ренормализационная группа → ФРАКТАЛЬНОСТЬ HBP**
- Scale invariance: паттерны на микроуровне (нейронные ансамбли) воспроизводятся на макроуровне (поведение)
- Fractal brain: масштабная инвариантность в структуре и динамике мозга
- → Обоснование: ригидный аттрактор виден на всех масштабах; фрактальная вложенность уровней HBP

**8. Теория информации (Шеннон) → ЭНТРОПИЯ И НЕГЭНТРОПИЯ**
- Entropic Brain Hypothesis: чем выше ригидность аттрактора → тем ниже информационная энтропия → сужение бытия
- Synergistic workspace for consciousness (eLife 2024): синергия FPN + DMN
- → Количественная мера негэнтропии в термодинамическом столпе HBP

**9. Стохастический резонанс → ОБОСНОВАНИЕ DISRUPT-ФАЗЫ**
- Оптимальный уровень шума улучшает работу нелинейных систем (inverted U)
- MBA-модель: низкий DA → потребность в большем шуме для оптимальной когниции
- → Терапевтическая интервенция должна попасть в «зону резонанса»: достаточный PE для обновления модели, но не разрушительный

**10. Big Data & Predictive Analytics → T-HBP & TRACK**
- ML предсказывает депрессию с точностью 86%+ (Random Forest, 2025)
- PETRUSHKA (2025): персонализированный выбор антидепрессантов через big data + ML
- AI in MH market: $1.3B (2024) → $15.1B (2035)
- → Инструменты для T-HBP, Track-фазы, M-HBP

#### B. НЕЙРОНАУКИ И БИОМЕДИЦИНА

**11. Нейрохимия предикции → DA/5-HT DISSOCIАЦИЯ**
- Мета-анализ 107 исследований (2025): DA = reward prediction errors (approach); 5-HT = punishment prediction errors (avoidance/correction)
- DA и 5-HT отслеживают социальный контекст в substantia nigra (Nature 2024)
- → Нейрохимическая «прошивка» prediction error; обоснование Hardware failure

**12. Интероцепция и аллостаз → МОСТ FM2 ↔ FEP**
- Interoceptive energy allostasis (Science Advances, 2025): дисрегуляция генерирует коморбидность
- PAIO + INT framework (Ibanez & Northoff, 2024): аллостатическая интероцептивная перегрузка
- Bayesian interoceptive precision + anxiety (2024): вычислительные параметры снижают тревогу
- → Конкретный механизм, связывающий стресс, Hardware и PE

**13. Нейропластичность и нейрогенез**
- Karolinska Institutet (2025, Science): подтверждён нейрогенез в гиппокампе 0–78 лет
- Незрелые нейроны: сниженный порог для LTP
- → Биологическое обоснование: фазовый переход возможен на уровне субстрата; терапия создаёт новые нейронные элементы

**14. Ось кишечник-мозг (MGB axis)**
- Масштабный обзор (2025): 4 пути влияния — нейральный (вагус), иммунный (цитокины → нейровоспаление), эндокринный (HPA → кортизол), метаболический (SCFAs → барьерная дисфункция)
- → Расширение Hardware failure: субстрат ≠ только мозг; потенциал для S-HBP

**15. Поливагальная теория — ДИСКУССИЯ 2026**
- 39 исследователей: PVT «untenable» (2026)
- Porges & Polyvagal Institute: подробные таблицы, опровергающие критику
- → Рекомендация: нейроцепция и иерархия автономного ответа = клинически полезная эвристика; PVT в столпах HBP с осторожной формулировкой

#### C. ТЕОРЕТИЧЕСКИЕ РАМКИ

**16. Диссипативные структуры (Пригожин) → МОСТ ПРИГОЖИН ↔ FEP**
- Goldbeter (2018): бистабильность, осцилляции, пространственные паттерны, волны — 4 типа диссипативных структур в биологии
- 2025: нейронное обучение = самоорганизация диссипативной системы, минимизирующей свободную энергию
- Self-organized integration vs disintegration (2025): синергетика Хакена + Пригожин
- → Мозг = диссипативная структура, поддерживающая NESS; прямой мост Пригожин–Рыжов–FEP

**17. 4E когниция → БЫТИЕ = СИСТЕМА ОТНОШЕНИЙ**
- Embodied, Embedded, Enacted, Extended → FEP как математический мост
- → Расширенная когниция → FM5 (Бытие-в-цифре); воплощённая → HW↔SW; встроенная → Input failure

**18. Автопоэзис (Maturana/Varela, Luhmann) → P1 НА БИОУРОВНЕ**
- Организм конструирует реальность; автопоэтическая система самовоспроизводится, включая патологические паттерны
- Luhmann: система/среда как самосоздающееся различение
- → Обоснование P1; психика = автопоэтическая система, воспроизводящая себя (включая ригидный аттрактор)

**19. Теория процесса / EFT (Greenberg) → DISRUPT-ФАЗА**
- Эмоции = двигатель изменений, не зависимая переменная
- Маркер-управляемые задачи: эмоциональные процессы разворачиваются в последовательностях шагов
- → Конвергенция с Disrupt-фазой D-HBP; Greenberg + Bugental + FEP = триангуляция

**20. Теория относительности → МЕТАФОРА, НЕ ФОРМАЛИЗМ**
- Psychological Spacetime model (Zakay, 2016): замедление субъективного времени при ускорении движения
- → Каждый субъект проживает своё пространство-время; риторическая рамка, не формальная аналогия
