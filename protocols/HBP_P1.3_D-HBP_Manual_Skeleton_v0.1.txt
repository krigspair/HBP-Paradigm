## ПРЕАМБУЛА

D-HBP — основной терапевтический протокол HBP для индивидуальной
работы с депрессией и тревогой. Основан на пяти фазах,
каждая из которых операционализирует ключевые постулаты HBP.

Целевая аудитория мануала:
- Исследователи (Phase 1-2 SCED и групповые пилоты)
- Клиницисты, прошедшие HBP-обучение (Level A-B)
- Супервизоры D-HBP

Версия 0.1 — скелет (структура + ключевые компоненты);
полный мануал — v1.0 (после Phase 1 SCED).

## 1.1 Ключевые постулаты, операционализированные в D-HBP

D-HBP непосредственно реализует следующие постулаты HBP:

| Постулат | Операционализация в D-HBP |
|----------|--------------------------|
| П1 (Процесс) | Работа с динамикой (не «чертами»); EMA tracking |
| П3 (Аттракторы) | Картирование аттракторов Фазы 2; attractor disruption Фазы 3 |
| П4 (Фазовые переходы) | CSD-мониторинг; sudden gain как целевой outcome |
| П5 (Precision) | Выбор интервенции → рабочей гипотезе о precision weights |
| П6 (ET) | TTOM-driven assessment Фазы 1; ET-work Фаз 2-5 |
| П7 (TTOM) | Формальная оценка TTOM-S в начале каждой фазы |
| П8 (AISI) | Сессии спроектированы как AISI-opportunities |
| П9 (Rupture-repair) | Rupture protocol → активная repair sequence |
| П11 (CFR) | CFR profile → адаптация интенсивности и формата |
| П12 (Прагматизм) | Каждая интервенция оценивается клинически + данными |

## 1.2 Отличия от CBT (для позиционирования в KC-5)

| Параметр | CBT | D-HBP |
|----------|-----|-------|
| Единица работы | Мысль/убеждение | Аттрактор/паттерн прецизирования |
| Цель изменения | Когнитивная реструктуризация | Фазовый переход (PE-update) |
| Роль отношений | Терапевтический альянс (necessary) | AISI-среда (necessary + sufficient) |
| Мониторинг | PHQ-9 (categorical) | EMA (continuous) + CSD-alert |
| Timing | Fixed session structure | Adaptive (CSD-driven) |
| Epistemic trust | Implicit | Explicit (TTOM-S assessment) |

## 2.0 Обзор фаз

| Фаза | Название | Сессии | Основная задача |
|------|----------|--------|----------------|
| 1 | Assessment & Alliance | 1-3 | Диагностика + основание ET |
| 2 | Attractor Mapping | 4-8 | Картирование аттракторной структуры |
| 3 | Disruption & PE-work | 9-16 | Дестабилизация аттракторов; PE-опыт |
| 4 | Transition & Integration | 17-22 | Фазовый переход; new attractor |
| 5 | Consolidation | 23-25 | Закрепление; relapse prevention |

Общая длительность: 22-28 сессий (50 мин) + 2-3 booster сессии
(через 1, 3, 6 месяцев post-therapy).
Частота: 1-2 раза в неделю (Фаза 1-3); 1 раз в 2 недели (Фаза 4-5).

Adaptation rule: Если CFR profile (Phase 1) показывает Low Complexity,
длительность Фазы 2-3 сокращается (до 15 сессий total);
если High Complexity — увеличивается (до 35 сессий total).

## 2.1 ФАЗА 1: ASSESSMENT & ALLIANCE (Сессии 1-3)

### Цель
Установить epistemic trust + получить диагностическую картину
(аттракторная структура, CFR profile, Hardware factors).

### Компоненты

**Сессия 1: First Contact**
- Задача терапевта: создать AISI (authentic interest +
  social intelligence + идентификация с переживанием клиента)
- НЕ давать психоедукацию в С1 — слушать, отражать, validirovать
- Конец С1: краткое резюме «Что я услышал» (техника TTOM-seed)
- Домашнее задание: EMA setup (приложение + инструкция)

**Сессия 2: Assessment Deep Dive**
- TTOM-S (заполняется терапевтом после С1; обсуждается с клиентом в С2)
- Оценка Hardware factors (анамнез: эпигенетические маркеры,
  хронические воспалительные паттерны, if clinically relevant)
- Первичная гипотеза об аттракторной структуре (3-5 аттракторов)
- PHQ-9 + GAD-7 + ECR-12 (базовые измерения)

**Сессия 3: Contract & Shared Map**
- Psychoeducation: объяснение модели (без технического жаргона)
  «Мы будем отслеживать паттерны, которые Вас затягивают..."
- Shared attractor map: клиент и терапевт совместно рисуют
  предварительную карту аттракторов
- Therapy contract: цели, частота, EMA-commitment, rupture protocol

### Ключевые инструменты Фазы 1
- TTOM-S (терапевт-рейтинг epistemic trust)
- PHQ-9, GAD-7, ECR-12, WAI-SR (после С3)
- EMA setup (affect grid 5 items, 2x/day, 60 дней min)
- CFR initial estimate (qualitative, уточняется в Фазе 2)

### Переход в Фазу 2
Критерий: WAI-SR ≥ 40 (или клинический эквивалент).
Если WAI-SR < 40 после С3 → дополнительные 1-2 сессии Фазы 1
(max 5 сессий в Ф1 до reassessment).

## 2.2 ФАЗА 2: ATTRACTOR MAPPING (Сессии 4-8)

### Цель
Детальное картирование аттракторной структуры клиента;
начало EMA-driven мониторинга; уточнение CFR profile.

### Компоненты

**Attractor Identification Protocol (AIP):**
1. Narrative elicitation: «Опишите последний раз, когда
   Вы были в [депрессивном / тревожном] состоянии.
   Что его запустило? Что поддерживало?"
2. Temporal tracing: хронология последних 3 эпизодов
3. EMA review (first 2 weeks): паттерны в данных
4. Attractor labeling: клиент и терапевт совместно называют
   и описывают 3-5 аттракторов (пример: «Аттрактор Изоляции»,
   «Аттрактор Перфекционизма»)
5. Transition mapping: что переключает между аттракторами?

**CFR Profile Assessment:**
- EMA switching rate (первые 20 datapoints): estimate Flexibility
- Narrative evidence для Complexity и Resilience
- CFR Profile: Low/Medium/High для каждого компонента

**Network map (optional):**
- Если NCT доступен: формальная network analysis EMA данных
- Если нет: клиническая оценка connectivity между симптомами

### Домашняя работа (Ф2)
- EMA продолжается
- Журнал аттракторов: «Когда сегодня я замечаю Аттрактор [X]
  — запишите контекст (кто, где, что запустило)"

### Переход в Фазу 3
Критерий: 3-5 аттракторов идентифицированы и названы;
CFR profile установлен; EMA данные ≥ 14 дней.

---

## 2.3 ФАЗА 3: DISRUPTION & PE-WORK (Сессии 9-16)

### Цель
Дестабилизировать аттракторы через Prediction Error (PE) опыт;
создать условия для фазового перехода.

### Теоретическое основание
Аттрактор = устойчивое prior belief + поведенческий паттерн.
Изменение = PE-update: опыт, нарушающий prior prediction,
достаточно сильный для reconsolidation.

### Компоненты

**PE-Identification:**
Каждую сессию: «Что в этой неделе было НЕ таким, как Вы ожидали?
Даже маленькое?» → Amplify + Integrate

**Attractor Disruption Techniques:**

1. **Behavioral Experiments (HBP-adapted):**
   Не «проверка мысли» (CBT), а «вход в PE-зону»:
   клиент намеренно делает что-то, что должно вызвать
   предсказание аттрактора — и наблюдает реальный результат.
   Пример (Аттрактор Изоляции): позвонить одному человеку
   → predict: «будет хуже» → actual: ?

2. **Mismatch-Based Interventions (Ecker-adapted):**
   Step 1: активировать аттрактор (vivid recall)
   Step 2: ввести juxtaposition experience (что-то,
     несовместимое с prior belief аттрактора)
   Step 3: reconsolidation window (4-6 hrs) —
     terapeut отмечает сессию как «mismatch session»
     → EMA next day: следить за изменениями

3. **Rupture-as-PE:**
   Если rupture возникает в сессии → использовать как
   терапевтическую PE-возможность (не избегать!)
   Протокол rupture (см. 3.1 ниже)

4. **AISI Deepening:**
   В каждой сессии Ф3: терапевт активно демонстрирует
   «это не то, что ты ожидал от отношений»
   (authentic caring, non-judgemental curiosity)

**CSD Monitoring:**
- Еженедельно: EMA variance + lag-1 autocorrelation
- CSD-alert (variance OR autocorr увеличение > 1 SD
  за последние 7 дней): «система приближается к
  фазовому переходу» → не уменьшать интенсивность,
  а поддерживать и направлять

**Sudden Gain Protocol:**
Если PHQ-9 drop ≥ 5 баллов за 1 сессию:
  → Dedicated session: «Что произошло? Что изменилось?"
  → Закрепить изменение (не объяснять — исследовать)
  → EMA review: что изменилось в паттернах?

### 3.1 RUPTURE PROTOCOL

Trigger: терапевт замечает rupture сигналы:
  - Клиент избегает зрительного контакта
  - Прямое несогласие с интерпретацией
  - Запоздалая ответ на вопросы
  - Повышение формальности

Шаги:
1. Name it (мягко): «Я замечаю, что между нами сейчас
   что-то изменилось. Я прав?"
2. Validate: «Это важно — Ваша реакция на меня сейчас
   много говорит о том, как работает система"
3. Explore: «Что именно я сделал/сказал, что вызвало это?"
4. Repair: «Я понимаю. Я [исправляю/признаю/подтверждаю]."
5. Integrate: «Что этот опыт (rupture+repair) говорит
   нам о Вашем Аттракторе [X]?"

Post-rupture: TTOM-S reassessment на следующей сессии.

### Переход в Фазу 4
Критерий (одно из):
  (a) CSD-alert был зафиксирован + последующее снижение
      симптомов ≥ 30% от baseline
  (b) Sudden gain произошёл
  (c) Клиент сам сообщает о «quality shift» в ощущении себя

## 2.4 ФАЗА 4: TRANSITION & INTEGRATION (Сессии 17-22)

### Цель
Закрепить фазовый переход; помочь новому аттрактору
стать достаточно устойчивым; интегрировать изменения
в self-narrative.

### Компоненты

**New Attractor Development:**
- «Если старый Аттрактор [X] — это [описание],
  то новый паттерн пока выглядит как... Назовём его?"
- Behavioral consolidation: действия, укрепляющие
  новый аттрактор (не «поведенческая активация» CBT,
  а «аттрактор-инвестиции»)

**Integration Work:**
- Narrative integration: «История изменения» —
  клиент формулирует own account of change
- Identity update: «Кто Вы теперь в этом новом паттерне?"

**EMA Tapering:**
- Ф4: EMA 1x/day (снижение с 2x)
- Если EMA switching rate стабилизировался →
  подтверждение нового аттрактора

**Relapse Prevention Planning:**
- Early warning signs (персонализированные EMA markers)
- Re-entry protocol: как вернуться, если старый аттрактор
  «затягивает» снова
- Booster sessions plan (1, 3, 6 мес)

### Переход в Фазу 5
Критерий: PHQ-9 ≤ 9 + клиническое суждение терапевта;
или клиент готов к завершению.

---

## 2.5 ФАЗА 5: CONSOLIDATION (Сессии 23-25)

### Цель
Завершение терапии с сохранением достижений;
передача «терапевтических инструментов» клиенту.

### Компоненты

**Termination Ritual:**
- С23: «Что Вы уносите из этой терапии?"
- С24: «Что Вы скажете себе через год?"
- С25: Formal closure + EMA final review + final measures

**Self-Therapist Training:**
- Клиент обучается самостоятельному CSD-мониторингу
  (упрощённая версия: «замечать когда привычные мысли
  усиливаются — это сигнал")
- Self-rupture protocol: «Если в отношениях что-то
  нарушилось — как Вы теперь это замечаете?"

**Final Measures:**
- PHQ-9, GAD-7, WAI-SR, ECR-12, ETMCQ-R
- Client satisfaction (brief, 3 items)
- Therapist: TTOM-S final + case formulation update

## 3.1 Уровни подготовки

| Уровень | Требования | Может проводить |
|---------|------------|----------------|
| HBP-A (Foundation) | 2-дневный workshop + супервизия 5 случаев | Ф1-Ф2 |

| HBP-C (Supervisor) | HBP-B + 2 года + обучение супервизии | Супервизия + обучение |

## 3.2 Ключевые компетенции

**Домен 1: Теоретическая база**
- Понимание 5-фазной модели D-HBP
- Знание ключевых постулатов HBP (П1, П3-П9, П11-П12)
- Понимание EMA и CSD концепций (не обязательно математически)

**Домен 2: Клинические навыки**
- AISI-компетентность (аутентичный интерес, социальный
  интеллект, способность идентифицировать переживание)
- TTOM-S administration and interpretation
- Attractor Identification Protocol (AIP)
- Rupture detection and repair
- Mismatch-based intervention facilitation

**Домен 3: Мониторинг и адаптация**
- EMA setup и интерпретация
- CSD-alert recognition
- Adaptive protocol adjustment (CFR-driven)

**Домен 4: Этика и безопасность**
- Работа с суицидальностью в HBP-контексте
  (см. Chapter 4: Harm Protocol)
- Границы компетентности HBP-A vs HBP-B
- Когда направлять: коморбидность, psychosis, severe BPD

## 3.3 Супервизия

Формат супервизии D-HBP:
- Индивидуальная: 1x/2 недели для HBP-A;
  1x/месяц для HBP-B (ongoing)
- Групповая: 1x/месяц (peer case review)
- Format: live supervision (1-2 сессии/случай)
  или recording review (EMA + session notes + TTOM-S)

## 4.1 Суицидальность в D-HBP контексте

### Теоретическое основание
В HBP-рамке суицидальность = аттрактор высокой
устойчивости («Аттрактор Небытия» или «Аттрактор
Болевого Эскейпа»). Фазовый переход из него возможен,
но требует специальных условий.

### Уровни риска и ответ

**Уровень 0 (пассивные мысли, без плана):**
- D-HBP продолжается с усиленным AISI-фокусом
- EMA добавить: ежедневный пункт «интенсивность
  мыслей о смерти» (0-10)
- TTOM-S reassessment

**Уровень 1 (активные мысли, без плана/намерения):**
- Добавить crisis check-in (1x/week дополнительный контакт)
- Safety planning (стандартный + HBP attractor frame:
  «Что помогает перейти из аттрактора?"
- Консультация супервизора в течение 48 часов

**Уровень 2 (план или намерение):**
- СТОП D-HBP как основной модальности
- Кризисное направление (психиатр / кризисный центр)
- D-HBP может возобновиться после стабилизации (≥ 4 недели)

### Специфика D-HBP
В Фазе 3 (disruption) — повышенный мониторинг:
дестабилизация аттракторов может временно усиливать
суицидальные мысли («precritical period»).
Если в Ф3 появляются суицидальные мысли → немедленно
оценить уровень → при Уровне 1+ → пауза Ф3,
возврат в Ф2 (stabilization-focused).

## 4.2 Границы применения D-HBP

**Подходит для D-HBP (current version):**
- Депрессия (умеренная/умеренно-тяжёлая, PHQ-9 10-19)
- Тревога (умеренная, GAD-7 10-14)
- Смешанная депрессия/тревога
- Наличие идентифицируемых аттракторов

**Требует адаптации или направления:**
- Тяжёлая депрессия (PHQ-9 ≥ 20) → только HBP-B/C
- Активная суицидальность (Уровень 2) → психиатр первично
- Психоз → не показан
- Severe BPD → только HBP-C с адаптацией
- Активная зависимость → не показан без параллельной
  addiction treatment

## 5.2 EMA Protocol Detail

**5 items (2x/day, вечер + утро):**
1. «Насколько Вы чувствуете себя в ловушке сейчас?» (0-10)
2. «Насколько Вы чувствуете связь с другими?» (0-10)
3. «Насколько легко Вам переключиться сейчас?» (0-10)
4. «Насколько сильна тревога прямо сейчас?» (0-10)
5. «Насколько сильна тоска/грусть прямо сейчас?» (0-10)

CSD computation (еженедельно, автоматически):
- Variance (items 1+4+5)
- Lag-1 autocorrelation (items 1+4+5)
- Switching rate (item 3 > threshold transitions)

## 5.3 Case Formulation Template

D-HBP Case Formulation включает:
1. **Hardware Profile**: известные биологические факторы
2. **Input History**: травма, привязанность, среда
3. **Attractor Map**: 3-5 ключевых аттракторов + triggers
4. **CFR Profile**: C/F/R = Low/Med/High
5. **Epistemic Trust Level**: TTOM-S score + narrative
6. **Phase Target**: на какой фазе сейчас + что блокирует
7. **Kill Criteria Watch**: какие KC могут быть релевантны

APPENDIX A: ГЛОССАРИЙ D-HBP
========================================================================

| Термин | Определение в контексте D-HBP |
|--------|------------------------------|
| Аттрактор (psychological) | Устойчивый паттерн переживания/поведения, к которому система возвращается после возмущений |
| CFR Profile | Индивидуальный профиль по 3 осям: Complexity, Flexibility, Resilience |
| CSD-alert | Сигнал «критическое замедление» — variance+autocorr рост → близость к фазовому переходу |
| D-HBP | Depth-HBP — протокол индивидуальной терапии |
| EMA | Ecological Momentary Assessment — ежедневные замеры в реальных условиях |
| Epistemic Trust (ET) | Способность воспринимать информацию от значимого другого как достоверную и релевантную |
| Фазовый переход | Качественный сдвиг в аттракторной структуре (≠ количественное улучшение) |
| PE (Prediction Error) | Несоответствие между prediction и actual experience → движущая сила обучения |
| Rupture | Разрыв в терапевтическом альянсе, используемый как PE-возможность |
| TTOM | Therapist Theory of Mind — способность терапевта точно моделировать epistemic state клиента |
| TTOM-S | Шкала оценки TTOM (therapist-rated, 8 items) |

APPENDIX C: RESEARCH IMPLEMENTATION NOTES

Для Phase 1 (SCED N=6):
- Стандартизация: video recording всех сессий
- Adherence check: DPCCQ (Therapist Competence Questionnaire)
  или HBP-specific adherence checklist (Phase 0 разработка)

- Data management: EMA → REDCap или аналог

Для Phase 2 (групповой пилот):
- Рандомизация не требуется (pilot, pre-post)
- Control condition: waitlist or TAU (treatment as usual)
- Therapist training: минимум 2 терапевта HBP-B

Для Phase 4 (RCT):
- Control: CBT (manualized, matched sessions)
- Primary outcome: PHQ-9 + GAD-7 (8-week, 16-week)
- Secondary: EMA switching rate change, ETMCQ-R change
- Sample size: N=80 (40/arm), power 0.80 для d=0.50
