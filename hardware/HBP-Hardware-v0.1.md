## H-0. НЕЙРОМЕДІАЦІЯ ТА СИНАПТИЧНА ПЛАСТИЧНІСТЬ

### H-0.1 Огляд системи

**Нейромедіатори як "програмне забезпечення" мозку**: нейромедіатори — не просто "хімічні месенджери", а складна сигнальна система з множинними рецепторами, зворотними зв'язками та регуляторними механізмами. HBP-концепція "Hardware" охоплює цю систему як фундаментальний субстрат психологічних феноменів.

**Ключові системи** (для HBP-релевантності):
1. **Дофамінова система** — reward, motivation, prediction error
2. **Серотонінова система** — mood regulation, social behavior, interoception
3. **ГАМК/Глутамат** — баланс гальмування/збудження, тривога
4. **Норадренергічна система** — arousal, stress response, attention
5. **Ендоканабіноїдна система** — stress buffering, emotional memory

### H-0.2 Дофамінова система: деталізація

**Анатомія**: VTA → NAc (лімбічна), VTA → PFC (когнітивна), SN → стріатум (моторна). Для HBP критично: мезолімбічний та мезокортикальний шляхи.

**Reward Prediction Error (RPE)**[cite:1]: Schultz et al. (1997) — нейрони VTA кодують не саму нагороду, а різницю між очікуваним та отриманим. Це фундаментально для розуміння:
- Ангедонії: притуплений RPE-сигнал при депресії
- Адикції: гіперсалієнтність drug-cues
- Мотивації: RPE як "двигун" навчання

**FitzGerald et al. (2014)**[cite:2]: computational model дофаміну як Bayesian prior — мозок постійно оновлює "прогнозні моделі" через дофамінові сигнали. Імплікація для HBP: D-HBP може бути методом recalibration цих priors.

**Клінічна релевантність**:
- Депресія: знижена дофамінергічна активність у NAc → ангедонія
- Тривога: порушений RPE → надмірне передбачення загрози
- ПТСР: дисрегуляція reward/threat salience
- Шизофренія: гіпердофамінергія в стріатумі + гіподофамінергія в PFC

### H-0.3 Серотонінова система

**Анатомія**: ядра шва (raphe nuclei) → кора, гіпокамп, мигдалина, стріатум. 14+ типів рецепторів (5-HT1A-7B).

**Serotonin transporter (SERT) та ранній досвід**: Caspi et al. (2003)[cite:3] — взаємодія 5-HTTLPR (ген SERT) × ранній стрес. Короткий алель → підвищена реактивність на стрес. АЛЕ: пізніше — суперечливі реплікації. Для HBP-pragmatics: не "ген депресії", а "ген пластичності" (диференційна сприйнятливість).

**Серотонін та інтероцепція**: 95% серотоніну в кишечнику → зв'язок з Gut-Brain Axis (розділ H-5). Важливо: серотонін не лише "mood molecule", але й регулятор кишкової моторики та інтероцептивних сигналів.

**Обмеження monoamine hypothesis**: антидепресанти (SSRI) підвищують серотонін протягом годин, але клінічний ефект — тижні. Значить: механізм не прямий. Нові гіпотези: нейропластичність (BDNF), нейрогенез, зміни в HPA axis.

### H-0.4 ГАМК-глутаматний баланс

**E/I balance**: баланс збудження (глутамат) та гальмування (ГАМК) — критичний для нормальної роботи кори. Порушення → тривога, судоми, психоз.

**Кетамін як доказ**[cite:4]: швидкий антидепресивний ефект кетаміну (NMDA antagonist) → переосмислення depression як "glutamatergic disorder", не лише серотонінового. Glutamatergic модель депресії: надмірна активація NMDA → ексайтотоксичність → знижена BDNF → атрофія гіпокампу.

**ГАМК та тривога**: бензодіазепіни → GABA-A рецептори → анксіолітичний ефект. Але: tolerance, dependence. Альтернативи для HBP: дихальні практики → активація парасимпатики → підвищення ГАМК-тонусу природним шляхом.

### H-0.5 Нейропластичність

**BDNF (Brain-Derived Neurotrophic Factor)**[cite:5]: "fertilizer for the brain". Знижений при депресії, підвищується при:
- Фізичних вправах
- Антидепресантах
- Когнітивній стимуляції
- Соціальних контактах

**LTP (Long-Term Potentiation)**: молекулярний механізм навчання та пам'яті. AMPA receptor trafficking → синаптичне підсилення. Для D-HBP: терапевтичні сесії = структуровані можливості для LTP в терапевтично корисних нейронних мережах.

**Синаптична прунінг**: в адолесценції — масивне скорочення синапсів. Для HBP-розуміння: підліткові розлади як порушення прунінгу (шизофренія — надмірний прунінг у PFC).

## H-1. НЕЙРОІМУНОЛОГІЯ ТА ЗАПАЛЕННЯ

### H-1.1 Inflammatory hypothesis of depression

**Foundational evidence**: Musselman et al. (2001)[cite:6] — у пацієнтів з депресією підвищені IL-6, TNF-α, CRP. Мета-аналіз Dowlati et al. (2010): підвищені IL-6 та TNF-α при депресії (50+ досліджень).

**Sickness behavior model**: Dantzer et al. (2008)[cite:7] — cytokine-induced sickness behavior (fatigue, anhedonia, social withdrawal, hyperalgesia) нагадує симптоми депресії. Механізм: IL-1β, TNF-α → активація мікроглії → зниження серотоніну → IDO (indoleamine 2,3-dioxygenase) → kynurenine pathway.

**Kynurenine pathway**[cite:8]: триптофан → (IDO) → kynurenine → (різні шляхи):
- → Kynurenic acid (KYNA) — нейропротективний, антагоніст NMDA
- → Quinolinic acid (QUIN) — нейротоксичний, агоніст NMDA
При запаленні: зсув у бік QUIN → ексайтотоксичність → атрофія гіпокампу. Для HBP: психологічний стрес → запалення → нейротоксичність — фізичний механізм "психологічної травми".

### H-1.2 Мікроглія

**Мікроглія як "brain's immune cells"**: не лише захист від патогенів, але й:
- Синаптична прунінг (критично для розвитку)
- Нейротрофічна підтримка
- Регуляція нейрогенезу
- Surveillance мозкового середовища

**Microglial priming**: хронічний стрес → M1-поляризація мікроглії (прозапальна) → підвищена реактивність на наступні стресори. "Immune memory" мозку. Для HBP: раннє нехтування/травма → microglial priming → знижений поріг для депресивних епізодів у дорослому віці.

**Microglia та синаптична прунінг**: надмірна прунінг мікроглією → редукція синаптичних зв'язків. Патологічне при шизофренії, ASD, Альцгеймері.

### H-1.3 HPA-immune interactions

**Глюкокортикоїди як протизапальні**: кортизол → NF-κB ↓ → знижена транскрипція цитокінів. АЛЕ: хронічний стрес → глюкокортикоїдна резистентність → втрата протизапального контролю → хронічне запалення.

**Visceral fat та запалення**: ожиріння → адипокіни (leptin, resistin) + IL-6 від жирової тканини → системне запалення → neuroinflammation. Для HBP lifestyle interventions: фізична активність знижує не лише кортизол, але й системне запалення.

### H-1.4 Клінічні імплікації

**Inflammatory biomarkers як предиктори**: CRP > 3 mg/L → знижена відповідь на SSRI, краща відповідь на нортриптилін (TCA). Для HBP precision medicine: inflammatory subtype депресії може потребувати anti-inflammatory interventions (NSAIDs, omega-3, exercise).

**Psychotherapy та запалення**: Slavich & Irwin (2014)[cite:9] — CBT знижує запальні маркери. Механізм: зменшення threat appraisal → знижена sympathetic activation → менший прозапальний сигнал. Для D-HBP: терапія = anti-inflammatory intervention.

## H-2. НЕЙРОЕНДОКРИНОЛОГІЯ ТА СТРЕС

### H-2.1 HPA Axis: детальна анатомія

**Cascade**: Стресор → Гіпоталамус (CRH) → Передня гіпофіза (ACTH) → Надниркові залози (Кортизол). Кортизол → від'ємний зворотний зв'язок → гіпоталамус + гіпофіз + гіпокамп.

**Гіпокамп як регулятор HPA**[cite:10]: висока щільність глюкокортикоїдних рецепторів (GR) в гіпокампі → потужне гальмування HPA. АЛЕ: хронічний стрес → атрофія гіпокампу → втрата гальмування → гіперактивна HPA axis. Порочне коло: стрес → атрофія гіпокампу → більше стресу.

**Алостатичне навантаження (Allostatic Load)**[cite:11]: McEwen (1998) — кумулятивний ефект хронічного стресу на організм. Показники: кортизол, DHEA, артеріальний тиск, IL-6, HbA1c, BMI, etc. Для HBP: allostatic load = операціоналізація "накопиченого стресу".

### H-2.2 Cortisol awakening response (CAR)

**CAR**: 15-30 хв після пробудження — пік кортизолу (50-100% підвищення). Функція: підготовка до денних вимог, активація пам'яті, immune modulation.

**Порушення CAR**:
- Плоска CAR (відсутній ранковий пік) → burnout, хронічна втома, ПТСР
- Підвищена CAR → тривожні розлади, антиципаторна тривога
- Порушений ритм → circadian dysregulation

**Для D-HBP**: перевірка режиму сну/пробудження, оцінка щоденного стресу-напруги.

### H-2.3 Стрес та мозок

**Амигдала**[cite:12]: детектор загрози, модулятор консолідації пам'яті (адреналін + кортизол → enhanced memory for emotional events → ПТСР). Збільшена при тривожних розладах, ПТСР. Для HBP: fearful memories = overconsolidated emotional memories.

**Медіальна префронтальна кора (mPFC)**[cite:13]: регуляція амигдали, extinction learning. При депресії/тривозі: знижена mPFC → mPFC aktivness → слабка регуляція амигдали. Для D-HBP: CBT/ACT стратегії = тренування mPFC → extinction.

**Гіпокамп**[cite:14]: формування декларативних спогадів, просторова навігація, контекстуальна регуляція страху. Атрофія при депресії (8% зменшення об'єму) — частково оборотна при антидепресантах та фізичних вправах.

### H-2.4 Neurosteroids

**Allopregnanolone**: GABA-A позитивний модулятор, синтезується з прогестерону. Знижений при депресії, ПТСР. Флуктуації → передменструальний дисфорічний розлад (PMDD).

**DHEA (dehydroepiandrosterone)**: "anti-stress" гормон. DHEA:Cortisol ratio як маркер resilience. Знижений при burnout та хронічному стресі.

## H-3. ІНТЕРОЦЕПЦІЯ ТА ТІЛЕСНА СВІДОМІСТЬ

### H-3.1 Визначення та анатомія

**Інтероцепція**[cite:15]: Craig (2009) — "the sense of the physiological condition of the body". На відміну від екстероцепції (зовнішній світ) та пропріоцепції (положення тіла), інтероцепція = внутрішній стан організму.

**Анатомічний шлях**: Lamina I спинного мозку → NTS (nucleus tractus solitarius) → Parabrachial nucleus → Таламус (VPMpc) → **Острівкова кора (insula)** → ACC (anterior cingulate cortex).

**Інсула (insula)**[cite:16]: первинна інтероцептивна кора. Функції:
- Репрезентація фізіологічного стану
- Емоційне усвідомлення ("feeling states")
- Empaphy та social cognition
- Інтеграція тіло-свідомість

### H-3.2 Predictive Processing та інтероцепція

**Seth (2013) — "controlled hallucination"**[cite:17]: мозок постійно генерує predictions про стан тіла → порівнює з аферентними сигналами → prediction error → оновлення моделі або дія. Емоції = інтроцептивні predictions, не просто реакції.

**Implication для HBP**: патологічні емоції = помилкові інтероцептивні predictions. Тривога = prediction про небезпечний внутрішній стан без відповідної аферентації. Паніка = catestrophic misinterpretation of normal interoceptive signals.

### H-3.3 Інтероцептивна точність та психопатологія

**Порушення інтероцепції при**:
- Тривожних розладах: гіперінтероцепція (надмірна сaliency тілесних сигналів)
- Депресія: гіпоінтероцепція (відключення від тіла, "numbness")
- Alexithymia: знижена інтероцептивна awareness → труднощі в ідентифікації емоцій
- Розлади харчування: дисторсія інтероцептивних сигналів голоду/насичення

**Heartbeat detection task**[cite:18]: стандартний метод вимірювання інтероцептивної точності. Пацієнти з тривогою — підвищена точність; з депресією — знижена. Для HBP: interoceptive accuracy як діагностичний та терапевтичний маркер.

### H-3.4 Тілесно-орієнтовані інтервенції

**Mindfulness та інтероцепція**: Farb et al. (2013)[cite:19] — медитація збільшує товщину інсули → покращення інтероцептивної awareness. Для D-HBP: body scan, breathing practices = interoception training.

**Vagal tone та інтероцепція**: HRV (Heart Rate Variability) як маркер vagal tone. Висока HRV → краща emotional regulation, менша тривога. Дихальні вправи → підвищення HRV → покращення emotional regulation.

## H-4. ЕПІГЕНЕТИКА ТА МІЖПОКОЛІННА ТРАВМА

### H-4.1 Механізми епігенетичної регуляції

**ДНК-метилювання**[cite:20]: метильна група на цитозин (CpG сайти) → зазвичай репресія транскрипції. Стабільне, але не незмінне — може модифікуватися досвідом.

**Гістонові модифікації**: ацетилювання → активація; метилювання → активація або репресія залежно від сайту. Більш динамічні, ніж ДНК-метилювання.

**Non-coding RNA**: miRNA, lncRNA — регуляція генної експресії на post-transcriptional рівні. Роль у стресовій відповіді та передачі інформації між поколіннями.

### H-4.2 Ранній досвід та епігенетика

**Meaney et al. — licking/grooming (LG) модель**[cite:21]: новонароджені щури з високим LG (багато материнської турботи) → підвищена метилювання GR гена → краща HPA регуляція у дорослому віці → нижча реактивність на стрес. Cross-fostering: ефект залежить від прийомної матері (не біологічної) → behavior, не гени.

**NR3C1 (ген GR) метилювання у людини**[cite:22]: McGowan et al. (2009) — жертви дитячого насильства (суїцид) мають підвищене метилювання NR3C1 у гіпокампі → знижена GR експресія → дисрегульована HPA. Це молекулярний "відбиток" раннього нехтування.

**FKBP5 ген**[cite:23]: регулятор GR чутливості. FKBP5 × ранній стрес × генотип → підвищений ризик ПТСР. Yehuda et al. (2016): Holocaust survivors та їхні діти мають подібні FKBP5 methylation patterns.

### H-4.3 Міжпоколінна передача

**Трансгенераційна епігенетична спадковість**[cite:24]: Dias & Bhattacharya (2014) — миші, що відчували запах ацетофенону + шок, передавали Fear response на запах нащадкам через 2 покоління (F1, F2). Механізм: зміни в сперматозоїдах (olfactory receptor Olfr151 demethylation).

**Спірні питання**: більшість епігенетичних змін "стираються" під час гаметогенезу (epigenetic reprogramming). Трансгенераційна передача у людини — дискусійна, але:
- Holocaust study (Yehuda): погоджені FKBP5 метилювання у батьків та дітей
- Dutch Hunger Winter: нащадки мали підвищений ризик ожиріння, діабету
- Патернальний стрес: впливає на sperm miRNA → поведінка нащадків (у мишей)

**Для HBP**: "сімейні сценарії" та міжпоколінна травма можуть мати молекулярне підгрунтя, не лише психосоціальний механізм передачі.

### H-4.4 Епігенетична пластичність

**Reversibility**: Weaver et al. (2004) — введення TSA (HDAC inhibitor) дорослим щурам з Low-LG матерями → відновлення GR метилювання до рівня High-LG. Концептуально: "It's never too late" — епігенетичні зміни оборотні.

**Фізичні вправи та епігенетика**[cite:25]: аеробні вправи → деметилювання промоторів BDNF, FKBP5 → підвищена нейропластичність. Ще один механізм "exercise as medicine".

## H-5. GUT-BRAIN AXIS

### H-5.1 Анатомія та механізми

**Ентеральна нервова система (ЕНС)**[cite:26]: "second brain" — 500 мільйонів нейронів у кишечнику. Незалежна від ЦНС регуляція, але постійна двонаправлена комунікація через:
1. **Вагусний нерв** (80% аферентні волокна — від кишечника до мозку)
2. **Ентероендокринна система** (GLP-1, PYY, CCK → системний кровообіг)
3. **Імунна система** (мікробіота → GALT → цитокіни)
4. **Мікробні метаболіти** (SCFA, триптофан метаболіти → кровообіг → BBB)

**SCFAs (Short-Chain Fatty Acids)**[cite:27]: продукти ферментації клітковини мікробіотою. Butyrate, propionate, acetate:
- Джерело енергії для колоноцитів
- Epigenetic modulation (HDAC inhibition)
- BBB integrity підтримка
- Протизапальна дія
- Нейропротекція (butyrate → BDNF)

### H-5.2 Мікробіота та мозок

**Germ-free studies**[cite:28]: миші без мікробіоти → підвищена тривожність, порушена соціальна поведінка, гіперреактивна HPA axis. Recolonization → часткове відновлення нормальних параметрів. Докази причинності: мікробіота → поведінка.

**Fecal Microbiota Transplantation (FMT)**[cite:29]: Kelly et al. (2016) — FMT від депресивних пацієнтів до germ-free щурів → депресивно-подібна поведінка у щурів. Перший прямий доказ мікробіота → поведінка у людиноподібній моделі.

**Психобіотики**[cite:30]: Dinan et al. (2013) — пробіотики з ефектом на психічне здоров'я. Lactobacillus rhamnosus, Bifidobacterium longum — знижена тривога в дослідженнях на тваринах. Клінічні дослідження у людей: скромні, але обнадійливі ефекти при депресії, тривозі, IBS з коморбідною депресією.

### H-5.3 Триптофан-кинурениновий шлях у контексті мікробіоти

**Мікробіота → триптофан метаболізм**: мікробіота конкурує за субстрат з IDO/TDO шляхом. Dysbiosis → підвищена IDO активність → більше kynurenine → менше серотоніну. Для HBP: кишкова дисбіоз = один з механізмів серотонінового дефіциту.

**Serotonin production in gut**: 95% серотоніну синтезується в кишечнику (enterochromaffin cells). Мікробіота → стимулює цей синтез. Але: кишковий серотонін не проходить BBB → функції в периферичній нервовій системі та регуляції кишкової моторики.

### H-5.4 Клінічні імплікації

**Dietary interventions**[cite:31]: Mediterranean diet → краще психічне здоров'я (Jacka et al., SMILES trial). Механізм: висока клітковина → SCFAs, antioxidants → знижене запалення, omega-3 → нейропластичність.

**IBS-депресія коморбідність**: 50-90% пацієнтів з IBS мають тривогу/депресію. Chicken-and-egg: чи IBS викликає депресію через gut-brain сигнали, чи депресія → dysbiosis → IBS? Мабуть: обидва напрямки.

## H-6. GLYMPHATIC SYSTEM ТА СОН

### H-6.1 Відкриття гліфатичної системи

**Nedergaard Lab (2013)**[cite:32]: Xie et al. — сон активує гліфатичну систему мозку. Під час сну: міжклітинний простір збільшується на 60% → ліквор вимиває токсичні метаболіти (β-амілоїд, тау-протеїн).

**Механізм**: CSF надходить через периартеріальні простори → через AQP4 (aquaporin-4) канали астроцитів → переносить відходи до перивенозних просторів → виводиться через лімфатичні судини.

**Порушення при нестачі сну**: накопичення β-амілоїду → ризик Альцгеймера. Навіть одна ніч без сну → значний підйом Aβ42 у CSF.

### H-6.2 Сон та психічне здоров'я

**Bidirectional relationship**: нестача сну → психічні розлади; психічні розлади → порушення сну. Sleep disruption є одночасно:
- Симптомом (майже всіх психічних розладів)
- Причиною (загострення, тригер маній, збільшення суїцидального ризику)
- Терапевтичною мішенню (sleep hygiene як base intervention)

**REM сон та емоційна регуляція**[cite:33]: Walker et al. — REM сон = "overnight therapy". Під час REM: реактивація емоційних спогадів БЕЗ норадреналінового тону (амигдала + гіпокамп без NE) → знижена емоційна заряджень. Порушення REM (при ПТСР, депресії) → "накопичення" неопрацьованих емоцій.

### H-6.3 Порушення сну при психічних розладах

**Депресія**: ранні ранкові пробудження, порушений архітектура сну (знижена SWS, прискорений REM onset), зміни CAR. Sleep architecture як біомаркер депресії.

**ПТСР**: кошмари, hyperarousal, порушений REM. REM-сон без atonia (REM sleep behavior disorder) — пов'язаний з нейродегенерацією.

**Маніакальний епізод**: різке зниження потреби в сні — ранній ознака (часто за 1-2 дні до повного епізоду). Для D-HBP моніторинг: раптове зниження сну → alert.

### H-6.4 Circadian rhythms

**CLOCK гени**[cite:34]: CLOCK, BMAL1, PER1/2/3, CRY1/2 — молекулярний годинник. Порушення circadian rhythm (shift work, jet lag, нерегулярний сон) → підвищений ризик депресії, ожиріння, метаболічного синдрому.

**Social zeitgebers theory** (Frank & Kupfer): соціальні ритми (прийоми їжі, соціальна взаємодія, сон) синхронізують біологічний годинник. Disruption соціальних ритмів → circadian desynchrony → маніакальний/депресивний епізод при bipolardepression.

## H-7. МІТОХОНДРІАЛЬНА ТЕОРІЯ ДЕПРЕСІЇ

### H-7.1 Мітохондрії та нейрональна функція

**Енергетичні вимоги мозку**: 20% всієї енергії тіла при 2% маси. Нейрони = особливо енергозатратні клітини. Мітохондрії критичні для:
- ATP production
- Calcium buffering
- ROS management
- Апоптоз
- Синаптична пластичність (ATP для везикул, calcium для LTP)

**Mitochondrial dysfunction та психопатологія**[cite:35]: Gardner & Boles (2011) — "beyond the serotonin hypothesis": мітохондріальна дисфункція як загальний механізм депресії, тривоги, психозу. Докази:
- Знижена ATP у тромбоцитах та лімфоцитах при депресії
- Підвищена ROS при депресії та шизофренії
- Антидепресанти мають мітохондріальні ефекти

### H-7.2 Стрес та мітохондрії

**Glucocorticoids → мітохондрії**[cite:36]: хронічний кортизол → зниження мітохондріального мембранного потенціалу → знижена ATP продукція → клітинна дисфункція. Механізм: GR в мітохондріях → пряма регуляція мітохондріальних генів.

**Allostatic load → мітохондрії**: McEwen та Picard (2018) — мітохондрії як "cellular stress sensor". Accumulation of allostatic load = мітохондріальне зношення. Для HBP: психосоціальний стрес → фізичне старіння на клітинному рівні (мітохондрії).

### H-7.3 Лікування спрямоване на мітохондрії

**Фізичні вправи**[cite:37]: induces mitochondrial biogenesis (PGC-1α) → більше мітохондрій, кращий OXPHOS. "Exercise is medicine" — частково через мітохондріальний механізм.

**Ketogenic diet**: підвищення ketones → альтернативне паливо для мозку → bypassing glucose metabolism dysfunction → зменшення симптомів при treatment-resistant depression, depressionepilepsy.

**N-acetylcysteine (NAC)**: антиоксидант → glutathione підтримка → зниження ROS → нейропротекція. Обнадійливі дані при депресії, шизофренії, OCD.

## 9. СИНТЕЗ: HBP HARDWARE MODEL

### 9.1 Інтегративна модель

HBP Hardware = біологічний субстрат, на якому "запускається" психологічне програмне забезпечення. Ключові принципи:

**Принцип 1: Взаємна детермінація**
Мозок ↔ психіка — не односпрямовані, а взаємновизначаючі. Психологічні інтервенції змінюють мозок (нейропластичність, епігенетика). Мозкові стани визначають психологічну вразливість/стійкість.

**Принцип 2: Багаторівнева каузальність**
```
Гени × Епігенетика × Ранній досвід
        ↓
Нейробіологічні налаштування (HPA, нейромедіатори, мікробіота)
        ↓
Психологічні вразливості (когнітивні схеми, emotion regulation)
        ↓
Поведінкові патерни та симптоми
        ↓
Фідбек-петлі (поведінка → біологія → психологія)
```

**Принцип 3: Пластичність**
Jedes Niveau ist modifizierbar:
- Генний рівень: through epigenetic change
- Нейробіологічний: через ліки, вправи, психотерапію
- Психологічний: через навчання, терапію, досвід
- Поведінковий: через усвідомлені зміни

**Принцип 4: Часові масштаби**
- Гострий стрес: секунди-хвилини (amygdala, HPA surge)
- Хронічний стрес: тижні-місяці (hippocampal atrophy, microglial priming)
- Епігенетичні зміни: місяці-роки
- Трансгенераційні: покоління

### 9.2 HBP Hardware профіль пацієнта

Для кожного пацієнта D-HBP концептуально визначає:

**Inflammatory status**: ознаки хронічного запалення? (CRP, lifestyle factors)
**HPA calibration**: тип стресової реакції (hyper vs hypo HPA)
**Interoceptive accuracy**: зв'язок з тілом (alexithymia vs hypervigilance)
**Circadian health**: якість сну, режим, соціальні ритми
**Gut health**: дієта, IBS симптоми, antibiotics history
**Allostatic load**: накопичений стрес, кількість life events
**Epigenetic heritage**: раннє нехтування, attachment history
**Social connectedness**: ізоляція vs підтримка

### 9.3 Hardware-Software Interface

```
HARDWARE (біологія)          SOFTWARE (психологія)
──────────────────────────────────────────────────
HPA activation          ←→   Threat appraisal
Dopamine RPE            ←→   Motivation beliefs
Amygdala reactivity     ←→   Fear conditioning
mPFC activity           ←→   Cognitive reappraisal
Insula activation       ←→   Emotional awareness
Oxytocin release        ←→   Social trust schemas
Gut-brain signals       ←→   Visceral anxiety
Circadian rhythm        ←→   Behavioral activation
```

D-HBP interventions targeted at SOFTWARE level → change HARDWARE. і навпаки: HARDWARE interventions (lifestyle, medication) → підготовка "ґрунту" для SOFTWARE change.

## 10. КЛІНІЧНІ ІМПЛІКАЦІЇ ДЛЯ D-HBP

### 10.1 Біологічно-інформована психотерапія

**Psychoeducation про Hardware**:
- "Ваш мозок — не ворог, а орган, що намагається вижити"
- Нормалізація симптомів через нейробіологічне пояснення
- Зменшення self-blame: "Depression = inflammatory + neurobiological process, not weakness"

**Приклад CFR-техніки з Hardware-обгрунтуванням**:
> "Коли ви відчуваєте цю тривогу в грудях — це ваша інсула отримує сигнали від вагусного нерву. Ваш мозок інтерпретує ці сигнали як небезпеку. Давайте разом перевіримо: що відбувається насправді?"

### 10.2 Lifestyle as Medicine

**Ієрархія втручань** (evidence-based, знизу до гори):

1. **Сон** (фундамент): 7-9 год, стабільний режим, sleep hygiene
2. **Фізична активність**: 150 хв/тиж aerobic + 2x resistance training
3. **Харчування**: Mediterranean diet, клітковина, omega-3, fermented foods
4. **Соціальні зв'язки**: якість > кількість, real contact
5. **Stress management**: breathing, mindfulness, nature
6. **Cognitive engagement**: навчання, творчість, purpose

**Механістичне обгрунтування кожного**:
- Сон → гліфатичний клірінс + emotional consolidation + HPA reset
- Вправи → BDNF ↑ + запалення ↓ + мітохондрії ↑ + гіпокамп ↑
- Харчування → мікробіота ↑ + запалення ↓ + нейротрансмітери
- Соціум → oxytocin ↑ + запалення ↓ + HPA ↓

### 10.3 Medication Interface

**Для D-HBP практика**: розуміти психофармакологію не для призначення, а для:
- Пояснення пацієнту механізму дії
- Розуміння часових рамок ефекту
- Взаємодія терапії та ліків
- Розпізнавання, коли направити до психіатра

**Timing питання**:
- SSRI: 2-4 тижні до ефекту → patience + support через цей período
- Benzodiazepines: швидко, але не для тривалого використання
- Кетамін/esketamine: годинний ефект при TRD → нова ніша
- Mood stabilizers при bipolardepression: need strict adherence

### 10.4 Precision D-HBP

**Inflammatory subtype**: якщо є ознаки запалення (CRP > 3, ожиріння, хронічні болі, autoimmune) →:
- Пріоритет lifestyle interventions (anti-inflammatory)
- Omega-3 (2-4 г EPA+DHA)
- Розглянути N-acetylcysteine
- Collaborative care з лікарем

**HPA subtype**:
- Hyperactive HPA (підвищений кортизол): trauma-based → trauma processing
- Hypoactive HPA (знижений CAR, burnout pattern): gradual activation, pacing

**Interoception-based**:
- Alexithymia → body-oriented work first (somatic techniques)
- Hypervigilance → titrated exposure, не flooding

## 11. ВІДКРИТІ ПИТАННЯ ТА СУПЕРЕЧНОСТІ

### 11.1 Нерозв'язані питання

1. **Направлення причинності**: чи запалення викликає депресію, чи депресія → запалення, чи obидва — наслідки третього фактору (стресу, trauma)? Відповідь: мабуть, bidirectional + multiple pathways.

2. **Мікробіота-психіка**: сила ефекту у людини набагато менша, ніж у тварин. Клінічне значення пробіотиків при депресії — ще не доведено (якість досліджень низька).

3. **Трансгенераційна епігенетика у людини**: дуже важко відокремити від соціальної передачі травми. Методологічні обмеження.

4. **Kетамін**: механізм антидепресивного ефекту досі не повністю зрозумілий. NMDA antagonism? Sigma-1 receptors? Downstream AMPA? Synaptogenesis?

5. **Персоналізована медицина**: ми ще не вміємо надійно передбачати, який subtype у конкретного пацієнта.

### 11.2 Суперечності в літературі

**Serotonin hypothesis**: популярна стаття Moncrieff et al. (2022) "The serotonin theory of depression: a systematic umbrella review" → висновок: немає переконливих доказів, що серотонінова система відіграє роль у депресії. Контроверсія:
- Медіа: "SSRI myth exposed"
- Наукова відповідь: umbrella review має методологічні обмеження, SSRI мають доведену ефективність у RCT
- Для HBP: депресія ≠ "серотонінова недостатність", але це не означає, що SSRI неефективні

**5-HTTLPR та стрес**: Caspi et al. знаменита стаття → численні спроби реплікації → суперечливі результати → meta-analysis: ефект менший, ніж початково, але існує. Для HBP: обережність з strong genetic determinist claims.

### 11.3 Прогалини в HBP Framework

**Що ще потрібно вивчити для HBP**:
1. Нейробіологія специфічних психологічних конструктів HBP (Я-образ, etc.)
2. Нейронні корелати HBP-специфічних інтервенцій
3. Інтеграція circumplex model of affect з hardware
4. Нейробіологія духовних переживань (transcendence, purpose)
5. Cross-cultural variations в нейробіології психопатології

## 11.5 ДОДАТКОВИЙ МАТЕРІАЛ: Нові дані 2024-2025

*Ці факти з'явилися в пізніших DR-прогонах і доповнюють основні розділи:*

1. **Psylocibin та default mode network (DMN)**[cite:253]: Carhart-Harris et al. (2017, 2021) — псилоцибін знижує активність DMN ("ego dissolution") → переривання rumination cycles. Для HBP: DMN overactivity = neural substrate of self-referential rumination при депресії. CFR-техніка спрямована на той самий ефект через когнітивні means.

2. **Astrocytes у синаптичній передачі**[cite:254]: гліальні клітини = не просто "підтримка", а активні учасники триpartite synapse (пресинаптичний нейрон + постсинаптичний + астроцит). Астроцити регулюють glutamate clearance → ключові для E/I balance.

3. **Brexanolone (synthetic allopregnanolone)** FDA-approved 2019: rapid antidepressant effect (48-72 hours). Механізм: GABAergic potentiation + HPA modulation. Це підтверджує: depression не лише "monoamine deficiency", neurosteroids = окремий target. Потенціал для HBP: severe cases (postpartum, TRD) → розглянути neurosteroid augmentation.

4. **Glymphatic clearance як "brain hygiene"**: sleep = not passive, а active maintenance process. Імплікація: reframe sleep у D-HBP не як "rest", а як "hardware maintenance". CFR-техніка: "Ваш мозок потребує 7-8 годин для видалення токсичних білків — це biological necessity, not luxury."

5. **Meaney cross-fostering експерименти**: epigenetic changes reversible навіть після critical period. Імплікація: "It's never too late" — intensive therapy в дорослому віці може частково reverse early-life methylation patterns.

6. **Japanese longevity research + gut-brain axis**[cite:259]: centenarians мають preserved gut diversity + high SCFAs. Імплікація: gut health = not just "digestive issue", а fundamental для brain aging. D-HBP focus на elderly: dietary interventions (prebiotics, fermented foods) як adjunctive.
