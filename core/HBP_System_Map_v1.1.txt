РАЗДЕЛ I. ОНТОЛОГИЯ (П1–П3)
----------------------------------------------------------------------

П1. БЫТИЕ (Being-as-Process)
    Человек есть центр, процесс и способ проживания своего бытия.
    Бытие = живая, иерархически организованная система отношений
    субъекта с собой, другими, миром.

П2. СУБСТРАТ (Substrate)
    Бытие реализуется на физиологическом субстрате ЦНС.
    Субстрат — не «причина» бытия, а медиум.
    Мост к нейробиологии и психиатрии.

П3. ДВОЙНОЕ ОПИСАНИЕ (Dual Description)
    Каждый феномен описывается в двух регистрах:
    — феноменологическом (как переживается)
    — формальном (как моделируется)
    Ни один не редуцируется к другому.

----------------------------------------------------------------------
РАЗДЕЛ II. ФОРМАЛЬНЫЙ ЯЗЫК (П4–П5)
----------------------------------------------------------------------

П4. FEP = MODEL STRUCTURE (Construal)
    FEP используется как формальный язык, НЕ как онтология.
    FEP ≠ истина о мире; это конструал (Andrews, 2021).
    Конкретные параметры (learning rate, PE) — тестируемы;
    сам фреймворк — нет.
    Falsifiability firewall: каждый construal требует конкретной нуль-гипотезы.

П5. ACTIVE INFERENCE = ОПЕРАЦИОНАЛЬНАЯ МОДЕЛЬ
    Субъект минимизирует prediction error через действие и обновление модели.
    Единицы анализа: действие, восприятие, precision-weighting.
    EFE (Expected Free Energy) = exploration + exploitation.

----------------------------------------------------------------------
РАЗДЕЛ III. СОДЕРЖАТЕЛЬНАЯ МОДЕЛЬ (П6–П7)
----------------------------------------------------------------------

П6. ШЕСТЬ ИЗМЕРЕНИЙ БЫТИЯ (Dimensions of Being)
    Расширенная модель Лэнгле (4 FM) + 2 новых:
    1. Бытие-в-мире (Umwelt) — «Могу ли я быть?»
    2. Жизнь (Mitwelt) — «Нравится ли мне жить?»
    3. Самость (Eigenwelt) — «Имею ли я право быть собой?»
    4. Смысл (Überwelt) — «Ради чего я живу?»
    5. Бытие-в-цифре (Digitalwelt) — цифровая среда как 5-й мир
    6. Бытие-в-конечности-мира — экология, климат, цивилизационные риски

П7. ИЕРАРХИЯ ЗНАЧИМОСТИ (Мясищев)
    Система отношений имеет индивидуальную иерархию.
    Патогенны нарушения САМЫХ ЗНАЧИМЫХ отношений.
    Значимость — организующий принцип диагностики.
    Левин: поведение = f(личность, среда) — формула, конвергентная с П7.

----------------------------------------------------------------------
РАЗДЕЛ IV. ИНТЕРСУБЪЕКТИВНОСТЬ (П8–П9')
----------------------------------------------------------------------

П8. ТЕРАПЕВТИЧЕСКИЕ ОТНОШЕНИЯ
    Терапевт ≠ нейтральный экран.
    Отношения = second-order self + AISI (Active Intersubjective Inference).
    Терапевт моделирует клиента, клиент моделирует терапевта,
    оба обновляются.
    Reverie (Бион) = mode of being терапевта с максимальной
    чувствительностью к PE клиента.
    Риск Милгрэма: терапевт как авторитет → клиент делегирует
    active inference → опасность для agency.

П9. ЭПИСТЕМИЧЕСКОЕ ДОВЕРИЕ (Fonagy)
    Рабочие шкалы: Mistrust и Credulity (НЕ Trust; r = 0.12 — не работает).
    ETMCQ-R: 3-факторная модель (Trust, Mistrust, Credulity).
    Без эпистемического доверия обновление priors невозможно.
    Campbell et al. 2025, N=525: Mistrust r=0.41, Credulity r=0.36.

П9'. TTOM (Theory of Third-Order Mentalizing)
    Терапевт ментализирует то, как клиент ментализирует терапевта.
    Тройная рекурсия = минимальная сложность для глубинного изменения.

----------------------------------------------------------------------
РАЗДЕЛ V. МЕХАНИЗМЫ ИЗМЕНЕНИЯ (П10–П11)
----------------------------------------------------------------------

П10. MISMATCH → RECONSOLIDATION (Ecker)
    PE, доставленный в окно 4–6 часов, запускает реконсолидацию памяти.
    PE должен быть достаточно сильным, но не вызывать защитное отключение.
    PE ≠ инсайт; PE = переживаемое несовпадение (mismatch experience).
    AIMCT (Active Inference Model of Coherence Therapy) — формализация.

П11. СЕТЕВАЯ ДИНАМИКА
    Система отношений = сеть.
    Инструменты: Network Control Theory, controllability metrics,
    EMA-мониторинг, CSD (critical slowing down).
    Осторожность: CSD даёт ложные срабатывания (Helmich et al. 2024).

----------------------------------------------------------------------
РАЗДЕЛ VI. МЕТА-ПРИНЦИП (П12–П14)
----------------------------------------------------------------------

П12. НЕГАТИВНАЯ ДОКТРИНА / КАРТА ≠ ТЕРРИТОРИЯ
    FEP-агностицизм. Ни одна модель не тождественна бытию.
    Все формализмы — инструменты, не истины.
    Система должна быть способна отвергнуть собственные постулаты
    при столкновении с данными.

П13. PREDICTION ERROR КАК УНИВЕРСАЛЬНЫЙ МЕХАНИЗМ
    PE = способ, которым система обнаруживает расхождение между
    моделью и реальностью. PE пронизывает все уровни:
    от нейронного до экзистенциального.

П14. ЛЕСТНИЦА СЛОЖНОСТИ (пункт В мета-принципа)
    Парадигма подстраивается под человека, а не человек под парадигму.
    Три уровня батареи:
    A) Минимальный: клиническое интервью + MAIA-2 + ETMCQ-R (+ СЖО для D/E-HBP)
    B) Средний: + EMA + идиографическая сеть + HGF
    C) Исследовательский: + nctpy + психофизиология + фазовый анализ

ЧАСТЬ IV. ЭТИОЛОГИЧЕСКАЯ МОДЕЛЬ
========================================================================

Три оси патогенеза:
1. HARDWARE FAILURE (субстрат) — нейрологические, генетические,
   органические нарушения → C-HBP
2. INPUT FAILURE (среда) — травма, депривация, токсичная среда → B-HBP
3. SOFTWARE FAILURE (интериоризированный код) — дисфункциональные
   паттерны отношений, ригидные priors → D-HBP

Патогенез: психопатология = ригидный аттрактор (locked-in brain /
attractor trapping), сужающий бытие. Симптом — лучшее решение системы.

Уровни организации личности (МакВильямс, v1.1):
- Невротический = гибкий аттрактор, PE переносится, модель обновляется
- Пограничный  = полуригидный, splitting, PE вызывает дезорганизацию
- Психотический = locked-in, PE не различается, модель не обновляется

Психологические защиты (МакВильямс, v1.1):
= стратегии минимизации PE БЕЗ обновления модели (action без inference):
  проекция, рационализация, расщепление, отрицание, вытеснение,
  реактивное образование, интеллектуализация, сублимация и т.д.

Механизмы прерывания контакта (Перлз, v1.1):
= каталог стратегий обращения с PE на границе контакта:
  - Ретрофлексия: PE направлен на себя вместо среды
  - Проекция: свои priors приписаны другому
  - Конфлуэнция: стирание границы self/other → потеря PE
  - Дефлексия: отвод PE от зоны контакта
  - Интроекция: чужие priors приняты без переработки
  - Имплозия → Эксплозия: накопление → взрывной фазовый переход

ЧАСТЬ V. МОДЕЛЬ ЗДОРОВЬЯ
========================================================================

Здоровье = Complexity + Flexibility + Resilience (C+F+R)
- Complexity: многомерность, разнообразие состояний
- Flexibility: способность переключаться между аттракторами
- Resilience: способность возвращаться после дестабилизации

Конвергенция с Д.А. Леонтьевым (v1.1):
  Личностный потенциал (ЛП) ≈ F + R в одном конструкте:
  - Потенциал самоопределения → exploration (EFE)
  - Потенциал реализации → exploitation (EFE)
  - Потенциал сохранения → resilience

ЧАСТЬ XVII. ТЕРМОДИНАМИЧЕСКОЕ ЯДРО (Рыжов + Пригожин) [v1.1 NEW]
========================================================================

Б.Н. РЫЖОВ — Системная психология:
- Психика = открытая термодинамическая система
- Психологическая энтропия = мера неопределённости / дезорганизации
  системы отношений
- Негэнтропия = упорядочивание, структурирование, осмысление
- Конвергенция с Пригожиным: диссипативные структуры поддерживают
  порядок за счёт постоянного притока энергии

HBP-ИНТЕГРАЦИЯ:
- Здоровье (C+F+R) = низкая энтропия при высокой сложности (негэнтропия)
- Патология — два полюса:
  а) Рост энтропии → хаос, дезорганизация, кризис
  б) Падение энтропии до нуля → ригидный аттрактор, «мёртвый порядок»
- Терапия = управление энтропией: дестабилизировать ригидный аттрактор
  (↑ энтропию), затем помочь системе самоорганизоваться в новый порядок
  (↓ энтропию через негэнтропию)
- Фазовый переход = момент, когда система покидает старый аттрактор
  (CSD → sudden gain/loss)

СВЯЗЬ С FEP:
- Free energy ≈ upper bound on surprise ≈ информационная энтропия
- Минимизация F = минимизация энтропии = негэнтропия
- Формулы:
  F = D_KL(q||p) + H(q)  [variational free energy]
  S_thermo = k_B × S_info  [мост термо ↔ информация]
- Рыжов даёт ПСИХОЛОГИЧЕСКОЕ содержание тому, что Фристон описывает
  ФОРМАЛЬНО

ЛАКАН + КИБЕРНЕТИКА (Семинар II, 1954–55) [v1.1 NEW]:
- Бессознательное структурировано как язык = цепь означающих
  = вычислительная система
- Лакан читал Винера; схема L (4 позиции субъекта в коммуникации)
- HBP-перевод: кибернетика Лакана = прото-active inference
  на уровне символического. Означающее → прогноз → PE при
  столкновении с реальным (Real)

ЧАСТЬ XVIII. ГИПНОЗ И ИЗМЕНЁННЫЕ СОСТОЯНИЯ (теория) [v1.1 NEW]
========================================================================

КЛАССИЧЕСКИЙ ГИПНОЗ:
- Суггестия = прямая манипуляция precision-weighting:
  ↑ precision на словах терапевта, ↓ на собственных сигналах
- Транс = состояние с изменённым precision allocation

ЭРИКСОНОВСКИЙ ГИПНОЗ:
- Утилизация: любое поведение клиента = ресурс
  → «встречать систему там, где она есть» (≈ П1 + П14)
- Непрямая суггестия = создание контекста, в котором система
  сама реорганизует priors

СВЯЗЬ С HBP:
- Теория транса = модель precision-manipulation
- Не включён как клинический инструмент HBP, но даёт
  теоретическую базу для M-HBP (созерцательные практики)
  и P-HBP (REBUS-модель, precision reset при психоделиках)
